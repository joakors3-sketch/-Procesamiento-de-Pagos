Sistema de Procesamiento de Pagos

Descripción

Este proyecto consiste en una aplicación de consola desarrollada en C#.

El sistema permite registrar distintos métodos de pago, mostrar la información de cada operación, procesar pagos y calcular cargos adicionales o comisiones según el medio de pago utilizado.

Objetivos

- Registrar pagos realizados por distintos clientes.
- Mostrar la información de cada pago.
- Procesar operaciones de pago.
- Calcular costos adicionales o comisiones.
- Mostrar el monto final de la operación.
